## Spacing Scale

Declared values (must be multiples of 4, using `vp` for HarmonyOS):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4vp | Icon gaps, inline padding |
| sm | 8vp | Compact element spacing (e.g., between log lines) |
| md | 16vp | Default element spacing (e.g., screen margins) |
| lg | 24vp | Section padding (e.g., between control panel and console) |
| xl | 32vp | Layout gaps |
| 2xl | 48vp | Major section breaks |
| 3xl | 64vp | Page-level spacing |

Exceptions: none

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | System Background (`#FFFFFF` or dark mode eq) | Background, surfaces |
| Secondary (30%) | `#F1F3F5` (Light Gray) | Log console background, cards |
| Accent (10%) | `#007DFF` (HarmonyOS Blue) | Primary CTA ("Run Tests" button) |
| Destructive | `#E84026` (HarmonyOS Red) | Error logs, crash traces |
| Success | `#64BB5C` (HarmonyOS Green) | Passed test indicators |

Accent reserved for: Primary action buttons ("Run Tests").
